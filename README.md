# CircuitBuild AR

An augmented reality application that provides a virtual environment to 
practice and simulate electronic circuits.

## Features
- Visualize and interact with in-built circuit configurations in AR
- Adjust circuit parameters in real-time and observe simulation changes
- Beginner-friendly interface for learning circuit behavior hands-on

## Tech Stack
- Unity3D
- Vuforia (AR SDK)
- C#

## How to Run
1. Clone the repo
2. Open in Unity (version X.X)
3. Build and deploy to an AR-compatible Android/iOS device
4. Point camera at the Vuforia marker to launch the circuit environment
